# 开发技术栈
Vue3 TS mock.js element-puls vite 
pinia axios  vue-router tailwind vueuse
# 小工具
echarts v-dollar 
# 项目介绍
  一个用mock.js模拟登录数据的权限管理商家后台管理RBAC模板
  三个用户角色 /admin common employee/   用户名同名,密码随意,
  路由根据登录用户角色生成渲染
  简单颗粒权限使用v-if="role[]"
  首页使用echarts渲染SVG数据展示  
  CRUD 在/order/order中体验
  项目响应式布局,适配各类尺寸
